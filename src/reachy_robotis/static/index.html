How to use

What this app does

1. Try an utterance

Open the dashboard and type “push the box”, “start OMY leader mode”, or “start AI Worker” in the Intent Resolver panel.

2. Inspect the action

The Action Catalog shows trigger phrases, target device, and execution path before anything runs.

3. Run safely

Only registered tasks and allowlisted command keys can execute. The LLM cannot invent shell commands.
